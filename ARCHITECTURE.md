- **ORM**: Entity Framework Core 8.0
- **Database**: SQL Server
- **Authentication**: JWT Bearer Token
- **Real-time**: SignalR
- **Validation**: FluentValidation
- **Mapping**: AutoMapper

- **Framework**: Angular 18
- **UI Library**: Angular Material
- **CSS**: Tailwind CSS

- **HTTP Client**: Angular HttpClient

- **Orchestration**: Docker Compose
